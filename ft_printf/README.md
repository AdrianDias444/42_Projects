## Description
This project consists of reimplment the function printf, using <stdarg> library for use variable arguments. This function support this kind of formats %(c,s,d,i,u,p,x,X,%)

## Instructions
Compile with standart flags(-Wall -Wextra -Werror) my 'ft_printf.c ft_printf_utils.c', or just use command make, for create 'libftprintf.a' for compile with the main file.c, 'CC ft_printf.c libftprintf.a and compare the output of the original printf with my function.
